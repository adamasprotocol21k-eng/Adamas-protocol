# 💎 ADAMAS PROTOCOL
**The Ultimate Web3 Gaming & Staking Ecosystem on Polygon Amoy**

ADAMAS PROTOCOL एक प्रोफेशनल क्रिप्टो-गेमिंग प्लेटफॉर्म है जहां यूजर्स गेम खेल सकते हैं, क्विज हल कर सकते हैं और अपने ABP पॉइंट्स को स्टेक करके पैसिव इनकम कमा सकते हैं।

## 🚀 Key Features
* **Neural Mining:** गेम खेलकर और माइनिंग करके ABP पॉइंट्स कमाएं।
* **Staking Vault:** 120% APY तक का रिवॉर्ड। (नियम: हर तीसरे क्लेम पर 24 दिन का लॉक)।
* **Knowledge Lab:** Maths और GK क्विज हल करें और रिवॉर्ड जीतें।
* **Wallet Integration:** MetaMask और Polygon Amoy टेस्टनेट के साथ पूरी तरह सुरक्षित।
* **Jackpot Lottery:** अपनी किस्मत आजमाएं और बड़े जैकपॉट जीतें।

## 📁 Project Structure
- `index.html` - लैंडिंग पेज
- `dashboard.html` - यूजर एक्टिविटी और डेली चेक-इन
- `staking.html` - स्टैकिंग और लॉक सिस्टम
- `games.html` - माइनिंग गेम हब
- `knowledge.html` - एजुकेशन और क्विज सेक्शन
- `wallet.html` - विड्रॉल (Min 5000 ABP)

## 🛠️ Technology Used
- **Frontend:** HTML5, CSS3 (Neon Gaming Theme), JavaScript (ES6+)
- **Web3:** Ethers.js
- **Network:** Polygon Amoy Testnet
- **Hosting:** Vercel
